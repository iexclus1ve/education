Существуют ли у курса какие-либо (предварительные) требования?

    Вы научитесь программировать на Python от простых, до весьма сложных задач, используя большое количество библиотек.
    Вам нужет только компьютер с Windows или Linux и интернет.

Для кого этот курс:

    Курс для самых начинающих, кто хочет стать программистом на языке Python, никакого опыта не нужно.
